# FitCheck (핏체크) — 빈티지/셀렉트숍 사이즈 핏 진단 도구

## 문제
> 빈티지/셀렉트숍 고객이 "이 옷 한 벌이 나한테 맞을까?"를 판단할 수 없어 구매를 망설이고, MD는 사이즈 문의에 수작업으로 대응하느라 시간을 낭비하는 문제를, 자동 핏 진단 웹 도구로 해결한다.

- **현재 상태**: 상품당 재고 1개(사이즈 하나)인 구조라 고객이 핏을 직접 판단해야 함. 사이즈 관련 문의가 전체 CS의 약 30% 차지. MD가 매번 실측 데이터를 수동으로 비교하여 답변하는 데 건당 5-10분 소요. 사이즈 불안으로 인한 이탈률 높음.
- **원하는 상태**: 고객이 상품 상세페이지에서 자기 신체 치수를 입력하면 부위별 핏 피드백("딱 맞아요" / "타이트해요" / "넉넉해요")을 즉시 확인. MD는 상품 실측만 등록하면 적합 체형 레인지가 자동 계산되어 노출.
- **성공 기준**: 사이즈 관련 CS 문의 50% 감소, 핏 진단 사용 고객의 구매 전환율 20% 향상

## 스킬
| # | 스킬명 | 한 줄 설명 | 상태 |
|---|--------|-----------|------|
| 1 | `/my-clarify` | 패션 MD의 모호한 기획전/사이즈 핏 요청 → 가설 기반 질문 → 구체적 실행 스펙 | ✅ 동작 |
| 2 | `/context-sync` | 흩어진 업무 컨텍스트(Slack, 노션 등) → 자동 동기화 → 통합 워크스페이스 | ✅ 동작 |

## 핵심 기능

### 기능 A — 상품 핏 레인지 표시 (MD/관리자용 → 고객 노출)
- 상품 실측 사이즈(어깨, 가슴, 허리 등)와 복종을 입력하면
- "이 옷은 키 160~168cm / 가슴둘레 82~90cm인 분에게 적합합니다" 같은 적합 체형 레인지를 자동 계산

### 기능 B — 고객 맞춤 핏 진단
- 고객이 자기 신체 치수(키, 몸무게, 가슴둘레 등)를 입력하면
- 부위별 핏 피드백 제공: 😊 딱 맞아요 / 🤏 타이트해요 / 📏 넉넉해요

## 변화 기록
- **Day 1**: "Claude Code가 뭔지도 몰랐다" → Claude Code 기본 기능 학습, 스킬/MCP 개념 이해
- **Day 3**: "내 업무에 맞는 Clarify 스킬을 직접 만들었다" → 패션 MD 특화 모호함 카테고리(테마/타겟/상품구성/가격전략/노출기간/성과지표) 설계 완료. FitCheck 프로젝트의 CLAUDE.md를 작성하여 AI 에이전트가 핏 계산 로직, 데이터 모델, UI/UX 가이드라인을 정확히 이해할 수 있는 Single Source of Truth를 확보.
- **가장 크게 달라진 점**: "사이즈 핏 정리해줘"라는 모호한 요청이 Clarify 스킬을 통해 "복종별 여유분 기준으로 부위별 핏 판정을 자동화하는 웹 도구"라는 구체적인 프로젝트 스펙으로 발전. CLAUDE.md에 핏 계산 공식(`(단면×2) - 고객둘레`)까지 명시하여, AI가 코드를 정확히 생성할 수 있는 수준으로 요구사항을 구체화함.
