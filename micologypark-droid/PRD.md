# AI 아침 컨텍스트 자동화 시스템

## 문제
> 매일 아침 Gmail + Notion을 수동 확인하느라 30분이 낭비된다

- **현재 상태**: 아침마다 Gmail 받은편지함, Notion 페이지를 각각 열어 수동 확인 — 약 30분 소요, 중요 항목 놓치는 경우 발생
- **원하는 상태**: 1주 후 아침 5분 이내에 전날 + 오늘 컨텍스트가 자동 정리된 문서를 받아보는 상태
- **성공 기준**: 컨텍스트 파악 시간 30분 → 5분 이내 단축, 주 5회 이상 스킬 활용

## 스킬
| # | 스킬명 | 한 줄 설명 | 상태 |
|---|--------|-----------|------|
| 1 | `/my-context-sync` | Gmail + Notion 병렬 수집 → 일일 컨텍스트 문서 자동 생성 | ✅ 동작 |
| 2 | `/session-wrap` | 작업 세션 내용 자동 요약 및 마크다운 저장 | ✅ 동작 |

- **가장 크게 달라진 점**: "무엇을 자동화할지 몰랐다" → "내 일상의 구체적 불편을 스킬로 직접 해결"
